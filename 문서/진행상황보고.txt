연구개발본부 사원 박대원입니다.

ITSK SmartMobility UI 변경 건 관련하여 주간 진행률 보고드리겠습니다.

1. 동작 시간 설정 기능 [완료] - 기준시간을 추가하여 1초 단위 외 다른 시간으로도 설정할 수 있도록 구현하였습니다.
	

2. 예약 기능 99% 완료되었습니다. -> 테스트 필요
	 
	[완료]
	1) 예약이 실행중일 때, 다른 동작/시나리오를 실행하면 실행되지 않도록 처리(웹 화면에서 alert를 띄워줍니다.)하였습니다.
	2) 예약이 실행되기 전, 다른 동작/시나리오를 실행시키려 할때, 예약 시간과 겹치게 되면 실행되지 않도록 처리하였습니다.
	3) 예약을 등록 후,  또 다른 예약을 추가 등록할 때 시간이 겹치면 등록되지 않도록 처리하였습니다.
	4) 서버가 꺼지게 되면 재실행 시 다시 예약목록을 스캔한 후 미실행된 예약들을 Timer에 재등록시켜 예약이 실행되도록 처리하였습니다.
	5) 예약 날짜는 정규표현식으로 형식에 맞지 않게 입력할 시 예약되지 않도록 예외처리하였습니다.

	[미완료]
	1) 예약이 실행되기 전, 다른 동작/시나리오를 실행시키려 할때, 예상 종료시간보다 10분정도의 term을 줘야합니다.
	2) 예약 기능 테스트를 진행한 후 결과를 정리하여 다음 주 화요일에 보고드리겠습니다.

3. x축 범위 조절 기능 [완료] -  [1초, 10초, 1분, 1시간] 4가지 단위를 콤보박스에 표현하며, 선택 시 그 단위로 조절됩니다. 

4.  종합 집계 데이터 확인 [완료] - '시작시간, 종료시간, 전체수행횟수, 성공횟수, 실패횟수, 최대응답시간, 최소응답시간' 을 통계조회 페이지에서 그래프 상단 테이블에 구현하였습니다.

5. 통계 그래프 페이지 구현 -
	1) [1,2,3 번 그래프] 전체 수행 횟수 ,성공횟수, 실패횟수 그래프 구현 - 완료
	2) [4번 그래프] 응답률 구현 - 미완료
	3) [5번 그래프] 응답시간 - 50% 완료
	4) [6번 그래프] 응답이 없는 시간 - 미완료
